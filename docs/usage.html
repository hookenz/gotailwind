Usage Guide

← Back to Home

Step 1: Create input.css

@import "tailwindcss";

Step 2: Run gotailwind CLI

go tool gotailwind --input input.css --output output.css

Step 3: Import output.css in your HTML

Use the generated CSS in your project like this:

<link rel="stylesheet" href="output.css">

Plugin Support

Tailwind plugins like daisyUI can be used with gotailwind, but they currently require manual download and import.

We’re exploring a future feature to support plugins automatically.

Notes

  • No need to install Node.js or Tailwind CLI separately
  • Automatically downloads and uses the correct Tailwind binary
  • Supports standard Tailwind CSS features and directives
