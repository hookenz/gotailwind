gotailwind

A small Go tool that makes Tailwind CSS easier to use in Go projects.

Quickstart

  1. Install via Go: 
    go install github.com/hookenz/gotailwind@latest
  2. Create a file named input.css with the following contents: 
    @import "tailwindcss";
  3. Run the CLI: 
    go tool gotailwind --input input.css --output output.css
  4. Include the output in your site: 
    <link rel="stylesheet" href="output.css">

See the full usage guide.

Why use gotailwind?

  • No need to install Node.js or npm
  • Uses official Tailwind CSS binary
  • Works well in Go development workflows
  • Perfect for lightweight setups and container builds
  • Supports Tailwind CSS v4+ features

Plugin Support

Plugins like daisyUI can be added by downloading them manually.

We’re exploring automatic plugin support in a future release.
