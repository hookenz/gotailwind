gotailwind

Run Tailwind CSS in Go projects without installing Node

go get -tool github.com/hookenz/gotailwind
GitHub Usage

Why gotailwind?

  • No need to install Node or Tailwind CLI manually
  • Automatically downloads the correct Tailwind binary
  • Supports standard Tailwind CSS features and directives

Quick Start

Create an input.css file with the following content:

@import "tailwindcss";

Then run the CLI tool to generate your CSS output:

go tool gotailwind --input input.css --output output.css

See the Usage page for full documentation.
